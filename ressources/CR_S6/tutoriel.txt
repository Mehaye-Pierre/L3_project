Bienvenue dans le tutoriel d'avalam.
Commençons par vous présenter le jeu :
Le jeu débute avec un plateau rempli de pions de deux couleurs, qui sont déplacable par chaque joueur.
Le but est de former des tours dont la couleur est determinée par le pion au sommet de cette tour, en les déplacant.

Dans ce tutoriel, vous jouerez les blancs.
On peut déplacer des pions en les positionnant au dessus d'autre pions, formant ainsi une tour. On ne peut pas déplacer un pion sur une case vide.
Les pions pouvant être déplacés sont entourés d'un léger halo **COULEUR**, et les endroits ou ils peuvent se déplacer sont entourés en **COULEUR** une fois le pion selectionné.

Tentez de déplacer quelques pions :
Commencez par bouger le pion blanc en haut à gauche sur son voisin de droite.

X 2B B
N X N
B N B

Bien ! notez que vous pouvez déplacer des pions de votre couleur, mais aussi les pions de la couleur de votre adversaire
Deplacez maintenant le pion noir à droite de la ligne centrale sur la tour que vous avez formé précédement.

X 3N B
N X X
B N B 

Vous pouvez déplacer une tour : créez une tour en deplaçant le pion noir de la ligne du bas sur le pion blanc à sa gauche, 
puis déplacez la tour ainsi formée sur le pont noir de la ligne centrale

X 3N B
3B X X
X X B

La hauteur maximale d'une tour est de 5 pions, il est donc impossible de former une tour de taille supérieure
La tour de la ligne centrale ne peut donc pas aller sur la tour de la ligne du haut.

Le jeu se termine lorsqu'il n'est plus possible de déplacer des pions, puis le décompte des points se fait.
Les joueurs marquent un point pour chaque tour étant surmontée d'un pion de leur couleur.

Finissons maintenant cette partie:
Déplacez le pion blanc en haut à droite sur la tour voisine.

X 4B X
3B X X
X X B

Vous avez trois tours de votre couleur, vous marquez ainsi trois points.
Les noirs ne amrquent pas de point, car ils n'ont aucune tour de leur couleur
Bravo ! vous avez gagné et ainsi terminé ce tutoriel.
